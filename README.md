# Jaha Koo Eel

A robotic remote controlled eel. A prop for the play [Haribo Kimchi](https://www.campo.nu/en/production/24342/haribo_kimchi).

A collaboration between [Adriaan Wormgoor](http://adriaanwormgoor.nl) and [Willem Vooijs](https://willem.vooijs.eu), produced for [Stichting Theater Utrecht](https://www.theaterutrecht.nl/innovationlab) and [Campo](https://campo.nu).
